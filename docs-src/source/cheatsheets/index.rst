Cheat Sheets (Acordeones)
=========================

A veces, al estar programando se nos olvida alguna instrucción, comando 
o combinación de teclas que nos brinda cierta funcionalidad. Podemos 
googlear o preguntarle a algún LLM la respuesta. Otra opción muy 
común es utilizar las *cheat sheets* (hojas de trampa), que también podríamos traducir 
como acordeones. Estas son hojas que podemos imprimir o tener a la mano para 
recordar los comandos más importantes. En la programación web en particular
tenemos que recordar y aprender a utilizar muchos comandos. Aquí están algunos
documentos que nos pueden ayudar:

- :doc:`linux`
- :doc:`ssh-keys`
- :doc:`postgresql`
- 🌐 `tmux <https://tmuxcheatsheet.com/>`_ 
- 🌐 `vim/nvim <https://vim.rtorr.com/lang/es_es/>`_
- `github <../_static/pdf/git-cheat-sheet.pdf>`_
